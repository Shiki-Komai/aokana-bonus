# SDB Server UI

## 概要
SDB (Spectral Defence Bureau) の内部サーバを想定したログイン + ダッシュボードUIです。1ページ構成で画面を切り替えます。

## 動作確認手順
1. Windows で `sdb_server.html` をブラウザ（Chrome / Edge / Firefox 推奨）で開く。
2. ロゴ画面をクリック（または5秒待機）してログイン画面へ遷移。
3. 自動入力された認証ID・パスワードのまま「ログイン」をクリック。

## フォルダ構成
- `assets/images/` : ロゴ画像
- `assets/files/` : ダウンロード用ファイル
